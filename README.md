# GhostNote

> Bringing visibility to your hidden work.

GhostNote is a lightweight Windows utility that helps engineers, IT professionals, and other technical workers capture meaningful work before it gets forgotten.

Instead of tracking productivity, GhostNote focuses on preserving the invisible work that often disappears between context switches, meetings, and interruptions.

---

## Why GhostNote?

Ever finished something important and then struggled to remember it during:

- Performance reviews
- 1:1s
- Promotion discussions
- Resume updates
- Weekly status meetings

GhostNote makes capturing that work take only a few seconds.

---

## Features

- Fast note capture
- Local SQLite database
- Search
- Date filtering
- Categories
- Edit existing entries
- Dark / Light themes
- Export (CSV / Markdown)
- Right-click desktop context menu
- Fully local (no accounts, no cloud)
