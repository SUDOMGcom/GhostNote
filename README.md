# SUDOMG GhostNote

> Bringing visibility to your hidden work.

GhostNote is a lightweight Windows application that helps engineers, IT professionals, automation specialists, and other technical workers quickly capture meaningful work before it gets forgotten.

Instead of tracking productivity, GhostNote focuses on preserving the invisible work that often disappears between context switches, meetings, interruptions, and busy days.

**Capture first. Organize later.**

## Why GhostNote?

Ever finished something important and then struggled to remember it during:

- Performance reviews
- 1:1 meetings
- Promotion discussions
- Weekly status updates
- Resume updates
- Interview preparation

GhostNote makes capturing those moments take only a few seconds.

## Features

- ⚡ Fast GhostNote capture
- 🖱️ Windows right-click desktop integration
- 📝 Edit existing GhostNotes
- 🔍 Search and filtering
- 📅 Date-based organization
- 🏷️ Categories
- 💾 Local SQLite database
- 📤 Export to CSV and Markdown
- 🌙 Dark and Light themes
- 🔒 Fully local — no accounts, cloud services, or subscriptions

---

## Privacy First

GhostNote was built to help **you** remember your work—not to monitor it.

- No telemetry
- No analytics
- No cloud sync
- No account required
- Your data stays on your computer

GhostNote is designed to reduce the friction of documenting meaningful work—not to measure productivity or watch employees.

## Philosophy

GhostNote is **not**:

- A time tracker
- Employee monitoring software
- Productivity scoring
- Surveillance software

GhostNote **is**:

- A personal capture tool
- A memory aid
- A record of meaningful work
